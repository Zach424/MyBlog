## 已知风险

- Vinext 仍需在项目范围内验证 Next.js 兼容边界。
- 当前修复版本的 Wrangler 干跑总上传为 `2645.68 KiB`，gzip 后 `625.60 KiB`；已低于项目预算，线上部署后仍需确认平台限制与冷启动表现。
- 动态主机元数据会让 Vinext 把 `/` 标记为 `Unknown` 路由；构建和 Worker 渲染均通过，部署验收时继续验证。
- Sites 首次生产版本中四类参数化详情路由统一返回 404，而同一构建在 Node 和本地 Workerd 正常；当前以显式路由包装规避，并用 Sitemap 覆盖门槛防止新增内容遗漏。
- 系统字体在不同操作系统上的字宽会略有差异，需要在浏览器验收中检查折行。
- 当前 CSP 为兼容 Vinext 的内联启动脚本与现有 CSS 保留 `script-src/style-src 'unsafe-inline'`；后续如框架支持 nonce，应继续收紧。
- GitHub 公开入口和首批内容已确定；最终域名、邮箱公开范围和其他联系方式仍待用户确认。
